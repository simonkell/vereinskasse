Diese Website

Diese statische Website setzt keine Analyse- oder Marketing-Cookies ein und bindet keine externen Schriftarten, Videos oder Trackingdienste ein. Beim Aufruf verarbeitet der Hostinganbieter technisch notwendige Verbindungsdaten, insbesondere IP-Adresse, Zeitpunkt, angeforderte Datei und Browserinformationen, um die Website auszuliefern und gegen Missbrauch zu schützen.

Hosting über GitHub Pages

Die Website wird über GitHub Pages bereitgestellt. Hinweise zur Verarbeitung durch GitHub finden sich in der Datenschutzerklärung von GitHub.
